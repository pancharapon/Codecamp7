selector example case

id use #id

class use .class

pseudo class use :

pseudo element use ::

attribute selector use tag[attribute (~, |, ^, $, *, none)= value]

combinator example

(,): selector1, selector2, ... , selector(n)

(space): selector1 selector2 ... selector(n)

(>): selector1 > selector2 > ... > selector(n)

(+): selector1 + selector2 + ... + selector(n)

(~): selector1 ~ selector2 ~ ... ~ selector(n)

(multiple): selector1 (, or space or > or + or !) selector2 (, or space or > or + or !) ... (, or space or > or + or !) selector(n)
